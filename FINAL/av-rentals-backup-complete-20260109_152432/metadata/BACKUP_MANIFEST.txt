RESTORE INSTRUCTIONS
=====================
1. Extract backup:
   cd /path/to/restore
   tar -xzf av-rentals-backup-complete-TIMESTAMP.tar.gz

2. Restore database:
   docker exec av-postgres-dev psql -U avrentals_user -d avrentals_db < database/av-rentals-db-TIMESTAMP.sql

3. Restore media files:
   cp -r media/* /path/to/storage/

4. Restore configuration:
   cp config/* /path/to/config/

5. Restart services:
   docker-compose restart
